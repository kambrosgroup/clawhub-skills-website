# Passive Income Skills Website

This is the website for the Passive Income Skill Stack - a collection of OpenClaw skills for freelancers.

## Live Site

https://kambrosgroup.github.io/passive-income-skills-website/

## Development

This is a static HTML site hosted on GitHub Pages.

To make changes:
1. Edit `index.html`
2. Commit changes
3. Push to GitHub
4. Changes deploy automatically

## Custom Domain (Optional)

To use a custom domain:
1. Update the `CNAME` file with your domain
2. Configure DNS with your provider
3. Enable HTTPS in GitHub Pages settings
